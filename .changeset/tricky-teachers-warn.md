Add more system fonts

Expands the metrics library to include more system fonts. Fonts added are:
- `Tahoma`
- `Lucida Grande`
- `Verdana`
- `Trebuchet MS`
- `Georgia`
- `Courier New`
- `Brush Script`

The library now support all the [Best Web Safe Fonts](https://www.w3schools.com/cssref/css_websafe_fonts.php).
